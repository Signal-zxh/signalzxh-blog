📝 网站介绍

这是一个基于 Go + Gin + MySQL + Redis 构建的全栈个人网站，包含文章管理、JWT认证、缓存机制和实用工具。

技术栈：

  • 后端：Go 1.24 + Gin
  • 数据库：MySQL 9.7
  • 缓存：Redis 7.2（文章详情缓存，10分钟过期）
  • 认证：JWT (golang-jwt/jwt/v5)
  • 前端：HTML + CSS + JavaScript（响应式设计）
  • 部署：Docker + Docker Compose

功能特性：

  • 📝 文章 CRUD（创建、阅读、更新、删除）
  • 🔐 JWT 安全认证
  • ⚡ Redis 缓存加速
  • 🍅 番茄钟专注工具
  • 📱 移动端响应式适配
